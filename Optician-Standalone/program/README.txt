Optician
--------

src contains the source code

generate-data.py {dir} generates data on the benchmarks in that directory

generate_IO_specs.py {dir} {n} generates n Flash Fill specifications of all test
files in that directory

generate_extraction_specs.py {dir} {n} generates n FlashExtract specifications
of all test files in that directory

example-inputs contains the benchmarks run

tests contains microbenchmarks and example-inputs

Optician is run as ./cmdline.native {testfile}
